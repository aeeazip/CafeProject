# CafeProject
웹프로그래밍 수업 과제를 확장하여 MVC Model1 구조를 따르는 카페 홈페이지를 제작

<br>

## 핵심 기능
- 카페 소개 및 메뉴판 페이지 제공
- 주문 및 리뷰 기능
- 장바구니 및 마이페이지 기능
- Q&A 기능
